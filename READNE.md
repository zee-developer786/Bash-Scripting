## !BASH SCRIPTING

#### 1: Command to check available shells

> `cat /etc/shells`

#### 2: Command to check Where Bash is located

> `which bash`

#### 3: To Create File

> `touch hello.sh`

#### 4: Command to check file permission

> `ls -al hello.sh`

#### 5: Command to give execution writes

> `chmod +x hello.sh`

#### 6: Command to execute file

> ` ./hello.sh`
